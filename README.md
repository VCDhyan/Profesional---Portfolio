
# VC Dhyan - Professional Portfolio

A high-performance, professional portfolio website for VC Dhyan, a Full-Stack Web Developer. This site is architected for speed, readability, and HR-focused engagement.

## Tech Stack

- **React 18+** for the UI framework.
- **TypeScript** for type-safe, maintainable code.
- **Tailwind CSS** for clean, utility-first styling.
- **Responsive Design** optimized for Mobile, Tablet, and Desktop.

## Key Sections

1. **Hero**: Value-driven headline for immediate impact.
2. **About**: Academic and professional background summary.
3. **Skills**: Categorized technical expertise for non-technical HR review.
4. **Projects**: Showcase of key work with "Team Lead" focus.
5. **Experience**: Quantifiable intern experience at Valise Technologies.
6. **Value Proposition**: "Why Hire Me" section focused on recruitment metrics.
7. **Contact**: Streamlined communication channels.

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Deployment

This project is ready for deployment on Vercel, Netlify, or GitHub Pages.

1. Build the production bundle:
   ```bash
   npm run build
   ```
2. Deploy the contents of the `dist/` or `build/` folder to your provider of choice.

## Design Philosophy

The site follows a "Corporate-Minimalist" aesthetic. It avoids heavy animations and distractions to ensure that the content—VC Dhyan's skills and experience—remains the primary focus for hiring managers and recruiters.
